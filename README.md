## ⭐Feature

### 1️. Home

<img width="100%" src="https://user-images.githubusercontent.com/31908501/226264950-29ddf767-3110-467e-886a-e2ab1900c561.gif"/>

- 항목 클릭 시 버튼에 해당 색이 그라데이션되어 보여지는 효과 구현
- 최초 접속 시 보이는 화면
- Skeleton UI를 이용한 상품 로딩 관리

### 2. SignUp

<img width="100%" src="https://user-images.githubusercontent.com/31908501/226267100-09959af5-927d-4581-a207-ffc62786529d.gif"/>

- Spinner를 이용한 회원가입 로딩 관리

- Spinner를 이용한 로그인 로딩 관리
